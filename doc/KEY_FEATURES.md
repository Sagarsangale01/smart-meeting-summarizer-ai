# 2. Key Features

- **AI Summary Generation**: Instant extraction of key insights from any text transcript.
- **Context-Aware (RAG)**: Uses past meeting data to maintain continuity across multiple sessions.
- **Section Regeneration**: Refine specific parts of a summary (e.g., just Action Items) without re-running the whole prompt.
- **Inline Editing**: Users can click on any AI-generated item to fix details manually.
- **Meeting History**: Local and backend persistence to revisit past summaries.
- **Export Options**: Export summaries to PDF or JSON for external use.
